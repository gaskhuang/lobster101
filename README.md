# 🦞 龍蝦101

龍蝦101 是一個以 **OpenClaw 自動化** 為主題的教學型商業網站：

- **首頁**：成交導向的漏斗首頁
- **案例庫**：118+ 成功案例的商業化整理
- **技能庫**：32 類 / 5,494+ Skills 的用途導覽
- **教案專區**：把案例拆成可上手的教學內容
- **服務方案**：從免費內容到企業導入與月費維護
- **諮詢入口**：直接導向 G大 的 Telegram / GitHub

## 網站結構

```text
/
├── index.html              # 首頁 / 漏斗頁
├── cases/index.html        # 案例庫
├── skills/index.html       # 技能庫
├── tutorials/index.html    # 教案專區
├── services/index.html     # 服務方案
├── contact/index.html      # 諮詢入口
├── assets/styles.css       # 共用樣式
├── assets/app.js           # 搜尋互動
├── cases/claw101-cases.md  # 原始案例整理
├── skills/skills-categories.md
└── .nojekyll
```

## 發佈到 GitHub Pages

1. 到 GitHub Repo `Settings` → `Pages`
2. Source 選 `Deploy from a branch`
3. Branch 選 `main`，Folder 選 `/ (root)`
4. 儲存後等待幾分鐘即可

## 網站定位

**不是單純資料庫，而是成交漏斗。**

使用者會從：

1. 看到案例 / SEO 內容
2. 建立信任
3. 自己試做教案
4. 卡住後轉諮詢
5. 進入代安裝 / 客製導入 / 月費維護
